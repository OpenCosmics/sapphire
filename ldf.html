3.6.3. Simulations based on theoretical models of EAS lateral distributions¶

class sapphire.simulations.ldf.BaseLdfSimulation(cluster, data, output, R, N, shower_size=63095.7344480193, **kwargs)¶

Simulation using a lateral distribution function as a model

Not using an EAS simulation but rather a lateral distribution function as the model for the detector simulation considerably speeds up the complete process. However, we will have to validate the use of an LDF.

This is the base class, so there is no LDF defined. Override calculate_ldf_value to complete an implementation.

run(positions=None, max_theta=None)¶

Run a simulation

This is the code which performs the simulation. It creates a list of positions, creates all necessary tables and performs the simulation.

Parameters:positions – if given, use these coordinates instead of generating new ones
